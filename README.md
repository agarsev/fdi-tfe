# TFE FDI UCM

Repositorio para cuestiones relativas a trabajos de fin de estudios (TFG, TFM)
en la Facultad de Informática de la Universidad Complutense de Madrid.

Dirigido especialmente a estudiantes de Antonio F. G. Sevilla <afgs@ucm.es>,
pero otros estudiantes y profesores bienvenidos.

Este repositorio refleja las opiniones de Antonio, en principio acorde a la
normativa escrita pero no necesariamente a los usos y costumbres de la facultad.
